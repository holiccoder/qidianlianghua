## 方案一：赛博朋克数据终端（Cyberpunk Data Terminal）

**设计运动**: 赛博朋克美学 + 数据可视化终端
**核心原则**:
1. 深邃的暗色基调，模拟高端交易终端
2. 霓虹色彩点缀（翠绿、电蓝、琥珀橙），营造科技感
3. 精密的网格系统，体现量化交易的精确性
4. 微妙的发光效果和数据流动感

**色彩哲学**:
- 背景：极深的蓝黑色 (#0a0e17)，不是纯黑，带有微妙的蓝色底色
- 卡片背景：深灰蓝 (#111827) 带半透明边框
- 主强调色：翠绿 (#00d4aa) - 代表增长和正向收益
- 次强调色：琥珀橙 (#f59e0b) - 代表重要数据和警告
- 信息色：电蓝 (#3b82f6) - 代表中性信息
- 负面色：珊瑚红 (#ef4444) - 代表亏损和风险

**布局范式**:
- 顶部固定导航栏，左侧LOGO + 中部标签导航 + 右侧状态指示
- 主内容区域采用响应式网格卡片布局
- 卡片内部采用左对齐的数据层次结构
- 底部信息区域采用全宽说明面板

**签名元素**:
1. 卡片边框带微妙的渐变发光效果（hover时增强）
2. 数据数字使用等宽字体，带有终端感
3. 小型图标徽章在卡片右上角

**交互哲学**: 精确、克制的交互反馈，hover时卡片边框微微发光，标签切换带平滑过渡

**动画**: 数字变化时的计数动画，卡片进入时的淡入上移，标签切换时的内容渐变

**字体系统**: 
- 标题：JetBrains Mono（等宽，终端感）
- 正文：Inter（清晰可读）
- 数据数字：JetBrains Mono（精确感）

## 方案二：极简暗黑仪表盘（Minimal Dark Dashboard）

**设计运动**: 瑞士国际主义 + 暗色模式
**核心原则**:
1. 极简主义，信息密度高但不拥挤
2. 严格的排版层次和留白控制
3. 单色为主，彩色仅用于关键数据
4. 卡片边框极细，依靠间距区分区域

**色彩哲学**:
- 背景：深灰 (#09090b)
- 卡片：略浅的灰 (#18181b) 带 1px 边框
- 主色：纯白文字 (#fafafa) 
- 绿色仅用于正向数据 (#22c55e)
- 红色仅用于负向数据 (#ef4444)
- 蓝色用于链接和操作 (#60a5fa)

**布局范式**:
- 极简顶栏，标签用下划线指示
- 内容区域采用等宽网格
- 大量留白，卡片间距宽松
- 数据用大号字体突出

**签名元素**:
1. 极细的分割线和边框
2. 大号数字配小号标签的对比
3. 状态指示用小圆点而非文字

**交互哲学**: 几乎无装饰的交互，hover仅改变透明度

**动画**: 极简过渡，仅有opacity和transform

**字体系统**:
- 全局：Space Grotesk（几何感强的无衬线）
- 数字：Tabular Lining数字

## 方案三：量化交易控制台（Quant Trading Console）

**设计运动**: 金融终端美学（Bloomberg Terminal 风格）+ 现代暗色UI
**核心原则**:
1. 深色背景模拟专业交易终端，但更现代化
2. 信息层次分明：大数字 > 标签 > 说明文字
3. 颜色编码系统：绿色=正向/运行，橙色=重要/警告，蓝色=信息，红色=负向
4. 卡片带有微妙的玻璃拟态效果和边框

**色彩哲学**:
- 背景：深海蓝黑 (#0b1120)，比纯黑更有层次
- 卡片背景：半透明深蓝 rgba(17, 25, 40, 0.75) 带模糊背景
- 边框：微妙的灰蓝 rgba(255,255,255,0.08)
- 翡翠绿 (#10b981)：价格、正向收益、运行状态
- 琥珀金 (#f59e0b)：关键数据、回购金额
- 天蓝 (#38bdf8)：信息标签、图标
- 珊瑚红 (#f43f5e)：负向数据、销毁

**布局范式**:
- 顶部导航：左LOGO区 + 中部pill形标签栏 + 右侧连接状态
- 主区域：自适应网格，4列→2列→1列响应式
- 卡片内部：图标徽章(右上) + 标签(左上) + 大数字(左中) + 说明(左下)
- 说明页：左右分栏，左侧介绍文字，右侧分配结构图

**签名元素**:
1. 卡片右上角的彩色圆形图标徽章（与参考截图一致）
2. 标签栏使用pill/胶囊形状，选中时带边框高亮
3. 百分比分配使用环形图或进度条可视化

**交互哲学**: 
- 标签切换带内容滑动过渡
- 卡片hover时边框颜色微微变亮
- 数字加载时有从0到目标值的计数动画
- 复制地址按钮带成功反馈

**动画**:
- 页面加载：卡片依次淡入（stagger animation）
- 标签切换：内容区域淡入淡出
- 数字：CountUp动画
- hover：边框发光 + 微妙上移

**字体系统**:
- 导航/标题：DM Sans（现代几何无衬线，粗体）
- 数据数字：JetBrains Mono（等宽，精确感，与参考截图一致）
- 正文说明：DM Sans Regular（清晰可读）
- 中文：系统默认（思源黑体/苹方）
